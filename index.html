Comment faciliter votre voyage de découvertes :

  1. Imprimez la grille de quadrillage proposée.
  2. Découpez les coordonnées imprimées.
  3. Disposez ces étiquettes numérotées sur votre plateau de jeu dans l'ordre indiqué.
  4. Commencez votre partie sur ce plateau enrichi des coordonnées.
  5. À chaque déplacement, il vous suffit de saisir les nouvelles coordonnées du quadrillage pour votre pion sur le site internet Monopstreet.
  6. Choisissez la thématique de découverte qui vous intéresse pour accéder à sa vue via Google Map.

Monopstreet est un moyen ludique de découvrir les richesses d’un quartier, aussi bien du point de vue culturel (monuments et bars-restaurants...) que commercial (magasins spécialisés, artisans, promotions...).
